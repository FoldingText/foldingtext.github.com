{{page.description}}

Marks the selected line as done, including a date-time stamp (in FoldingText's default date-time format) in the @done tag.

	@done(yyyy-mm-dd HH:MM:SS)
	
	@done(2012-12-05 21:15:34)

(Works with FoldingText's .todo mode)
